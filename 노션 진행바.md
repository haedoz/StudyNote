# 💻 노션 진행도 바 디자인 6종 모음 + 나만의 스타일로 커스터마이징 하는 법 | Notion

안녕하세요 :)
지난 포스팅에서 입력값 기반 진행도 바 수식을
공유해드렸었는데요!

생각보다 많은 분들이 관심 가져주셔서
오늘은 **같은 수식 하나로 만들 수 있는
다양한 진행바 디자인 6가지**를 모아서
공유해보려고 합니다

체크박스 기반 습관 진행바는 다음 편에서
이어서 다뤄드릴게요!

오늘 글에서 다룰 내용은 이렇게 정리돼요

- 수식 하나로 6가지 디자인 만드는 법
- 원하는 이모지 / 특수문자로 바꾸는 법
- `style()` 함수로 색상 입히는 법 + 사용 가능한 색상 총정리

---

## 진행바 마스터 수식

아래 수식 하나만 있으면
아이콘이랑 색상만 바꿔서 6가지 디자인을
전부 만들 수 있어요

속성 구성은 지난 글과 동일하게
**숫자 속성 2개(물 급수량 / 목표 급수량)**,
**수식 속성 1개(진행도)**로 만들어주시면 됩니다

```
let(
    /* 🎨 1. 진행바 디자인 설정 (이 부분만 수정하세요!) */
    fillIcon, "█",   /* 채워진 모양 */
    emptyIcon, "▒",  /* 비어있는 모양 */
    divider, " | ",  /* 텍스트 구분자 (예: |, ·) */
    /* 📊 2. 데이터 불러오기 */
    current, toNumber(prop("물 급수량")),
    target, toNumber(prop("목표 급수량")),
    /* ⚙️ 3. 진행바 계산 및 출력 (수정 불필요) */
    if(
        empty(target) or target == 0,
        "목표량을 먼저 설정해주세요",
        let(
            pRaw, current / target,
            p, min(1, pRaw),
            filledCount, floor(p * 10),
            bar,
            repeat(fillIcon, filledCount).style("blue") +
            repeat(emptyIcon, 10 - filledCount).style("gray"),
            remain, max(0, target - current),
            bar + " " +
            format(round(pRaw * 100)) + "%" + divider +
            format(remain) + "ml 남음"
        )
    )
)
```

포인트는 딱 3개예요

```
fillIcon, "█",   /* 채워진 모양 */
emptyIcon, "▒",  /* 비어있는 모양 */
divider, " | ",  /* 텍스트 구분자 */
```

이 세 줄만 바꿔주면 완전히 다른 느낌의
진행바가 만들어져요! 아래에서 이번에
새로 소개하는 6가지 디자인의 설정값을
그대로 알려드릴게요

## 오늘 공유하는 진행바 디자인 6종

| 디자인 | fillIcon | emptyIcon | 색상(style) |
|---|---|---|---|
| 미니멀 픽셀 네모 | `■` | `□` | blue |
| 원형 진행바 | `●` | `○` | red |
| 이모지 진행바 | `🧡` | `🤍` | 스타일 불필요 |
| 클래식 진행바 | `━` | `─` | 스타일 불필요(기본 검정) |
| 심플 블록 진행바 | `▓` | `░` | red |
| 라운드 블록 진행바 | `█` | `█` | 채운 부분 blue / 남은 부분 gray |

예를 들어 "원형 진행바"를 만들고 싶다면
마스터 수식에서 이 부분만

```
fillIcon, "●",
emptyIcon, "○",
```

이렇게 바꾸고, 색상 부분(뒤에서 설명)을
`"red"` 로 바꿔주면 끝이에요!

> 💡 이모지(🧡🤍)나 색이 이미 들어간 특수문자를
> 쓰실 때는 `.style()` 자체를 빼주시면 돼요.
> 이모지는 원래 자기 색을 갖고 있어서
> 따로 색을 입힐 필요가 없거든요!

---

## `.style()` 함수로 색상 입히는 법

특수문자(■, ●, ▓ 같은 도형 문자)로
진행바를 만들 때는 그 자체로는 색이 없기
때문에, 노션 수식의 `style()` 함수로
색을 입혀줘야 예쁜 컬러 진행바가 완성돼요

마스터 수식에서 색을 입히는 부분은 여기예요

```
repeat(fillIcon, filledCount).style("blue") +
repeat(emptyIcon, 10 - filledCount).style("gray"),
```

`.style("색상이름")` 형태로 작성하면
바로 뒤에 있는 텍스트에 색이 입혀지는
구조예요. 따옴표 안의 색상 이름만
바꿔주시면 됩니다!

### 사용 가능한 색상 리스트

**글자 색상**

```
gray / brown / orange / yellow / green / blue / purple / pink / red
```

**배경 색상**
글자 색상 뒤에 `_background`만 붙이면
배경색으로 적용돼요

```
gray_background / brown_background / orange_background /
yellow_background / green_background / blue_background /
purple_background / pink_background / red_background
```

**꾸미기 옵션(색상과 같이 조합 가능)**

```
b  → 굵게(Bold)
i  → 기울임(Italic)
u  → 밑줄(Underline)
s  → 취소선(Strikethrough)
c  → 코드 블록 스타일
```

예를 들어 진행이 100%가 됐을 때
글자를 굵고 초록색으로 강조하고 싶다면
이렇게 조합해서 쓸 수도 있어요

```
"완료!".style("b", "green")
```

색상과 꾸미기 옵션은 순서 상관없이
콤마(,)로 여러 개 같이 넣어도 됩니다!

## 오늘 공유한 디자인 커스텀 예시

**원형 진행바 + 빨간색**
```
fillIcon, "●",
emptyIcon, "○",
...
repeat(fillIcon, filledCount).style("red") +
repeat(emptyIcon, 10 - filledCount).style("gray"),
```

**심플 블록 진행바 + 빨간색**
```
fillIcon, "▓",
emptyIcon, "░",
...
repeat(fillIcon, filledCount).style("red") +
repeat(emptyIcon, 10 - filledCount).style("gray"),
```

**이모지 진행바 (색상 스타일 불필요)**
```
fillIcon, "🧡",
emptyIcon, "🤍",
...
repeat(fillIcon, filledCount) +
repeat(emptyIcon, 10 - filledCount),
```

## 🎁 보너스 디자인 5종 더 모아왔어요

이번에 소개한 6가지 말고도
바꿔서 쓰기 좋은 조합들을 더 챙겨왔어요!
마찬가지로 마스터 수식에서 3줄만
바꿔주시면 됩니다

| 디자인 | fillIcon | emptyIcon | 색상(style) |
|---|---|---|---|
| ⭐ 별점 진행바 | `★` | `☆` | yellow |
| 💗 핑크 하트 진행바 | `💗` | `🤍` | 스타일 불필요 |
| 💎 다이아 진행바 | `◆` | `◇` | purple |
| 🔋 배터리 진행바 | `▮` | `▯` | green |
| ▰ 그라데이션 블록 진행바 | `▰` | `▱` | blue |

예를 들어 "배터리 진행바"를 만들고 싶다면
이렇게 바꿔주시면 돼요

```
fillIcon, "▮",
emptyIcon, "▯",
...
repeat(fillIcon, filledCount).style("green") +
repeat(emptyIcon, 10 - filledCount).style("gray"),
```

## 🚦 응용편: 진행률 따라 색이 자동으로 바뀌는 진행바

지금까지는 색을 하나로 고정해서 썼는데요
진행률(%)에 따라 **색이 자동으로 바뀌는**
진행바도 만들 수 있어요!

- 30% 미만 → 빨간색 (아직 멀었어요)
- 30~70% → 주황색 (절반쯤 왔어요)
- 70% 이상 → 초록색 (거의 다 왔어요!)

```
let(
    fillIcon, "●",
    emptyIcon, "○",
    current, toNumber(prop("물 급수량")),
    target, toNumber(prop("목표 급수량")),
    if(
        empty(target) or target == 0,
        "목표량을 먼저 설정해주세요",
        let(
            pRaw, current / target,
            p, min(1, pRaw),
            filledCount, floor(p * 10),
            /* 🚦 진행률에 따라 색상을 자동으로 결정 */
            barColor, if(p < 0.3, "red", if(p < 0.7, "orange", "green")),
            bar,
            repeat(fillIcon, filledCount).style(barColor) +
            repeat(emptyIcon, 10 - filledCount).style("gray"),
            remain, max(0, target - current),
            bar + " " +
            format(round(pRaw * 100)) + "%" + " | " +
            format(remain) + "ml 남음"
        )
    )
)
```

포인트는 이 한 줄이에요

```
barColor, if(p < 0.3, "red", if(p < 0.7, "orange", "green")),
```

`p`(진행률)를 기준으로 조건을 나눠서
`barColor`라는 변수에 색 이름을 담아두고
`.style(barColor)` 형태로 적용하는 방식이에요

기준값(0.3, 0.7)이나 색상은
원하시는 대로 자유롭게 바꾸셔도 됩니다!
예를 들어 습관 관리용이라면
`0.5`, `0.9` 같은 식으로 조절해보세요 :)
